### Option 1: GitHub Pages (RECOMMENDED - Easiest)

1. **Create GitHub account** (if you don't have one): https://github.com/join

2. **Create new repository:**
   - Go to https://github.com/new
   - Repository name: `taniti-tourism`
   - Make it Public
   - Click "Create repository"

3. **Upload files:**
   - Click "uploading an existing file"
   - Drag and drop ALL the website files (html, css, js)
   - Click "Commit changes"

4. **Enable GitHub Pages:**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Under "Source", select "main" branch
   - Click "Save"

5. **Get your URL:**
   - Wait 2-3 minutes
   - Your site will be live at: `https://YOUR-USERNAME.github.io/taniti-tourism/`
   - **This is the URL you submit for your project!**

### Option 2: Netlify (Also Easy)

1. Go to https://www.netlify.com/

### Option 3: Vercel

1. Go to https://vercel.com/
